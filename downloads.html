Desenvolvimento vs produção Builds#

Nós fornecemos duas versões: uma versão React descompactado para o desenvolvimento e uma versão minified para a produção. A versão de desenvolvimento inclui avisos adicionais sobre erros comuns, ao passo que a versão de produção inclui otimizações de desempenho e tiras extras todas as mensagens de erro.

Se você está apenas começando para fora, certifique-se de usar a versão de desenvolvimento.

npm #

Recomendamos o uso de npm React com uma bundler como browserify ou webpack. Você pode usar os react e react-dom Pacotes. Após instalá-lo usandonpm install --save react react-dom, você pode usar:

var React = require('react');
var ReactDOM = require('react-dom');
ReactDOM.render(<App />, ...);

Cada um dos add-ons vive em seu próprio pacote.

Nota: por padrão, React será no modo de desenvolvimento. Para usar React no modo de produção, defina a variável de ambienteNODE_ENV de production (utilizando envify ou DefinePlugin de Webpack). A minifier que realiza eliminação de código de mortos como UglifyJSé recomendado para remover completamente o código extra presente no modo de desenvolvimento.
