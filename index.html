  2. Usando React a partir de npm

    Recomendamos o uso de React com um sistema de módulos commonJS como browserify ou webpack . Use os pacotes da  react e  react-dom npm

  3. // main.js  var React = require('react');  var ReactDOM = require('react-dom');    ReactDOM.render(    <h1>Hello, world!</h1>,    document.getElementById('example')  );  
  4. Para instalar React DOM e construir o seu pacote com browserify 
    $ npm install --save react react-dom babelify babel-preset-react  $ browserify -t [ babelify --presets [ react ] ] main.js -o bundle.js  
    Para instalar React DOM e construir o seu pacote com Webpack

    :

    $ npm install --save react react-dom babel-preset-react  $ webpack

    Nota:

    Se você estiver usando ES2015 , você vai querer usar também o

    babel-preset-es2015 package.

    Nota: Por padrão , React será no modo de desenvolvimento , que é mais lento , e não aconselhável para a produção. Para usar React no modo de produção , defina a variável de ambiente

    NODE_ENV to production (usando envify ou webpack's DefinePlugin). por exemplo:

    new webpack.DefinePlugin({    "process.env": {      NODE_ENV: JSON.stringify("production")    }  });

    Início Rápido Sem npm

    Se você não está pronto para usar npm no entanto, você pode baixar o starter kit que inclui cópias pré-construídos de React e React DOM.

    Download Starter Kit 0.14.6

    No diretório raiz do kit inicial , criar uma helloworld.html com os seguintes conteúdos .

    <!DOCTYPE html>  <html>    <head>      <meta charset="UTF-8" />      <title>Hello React!</title>      <script src="build/react.js"></script>      <script src="build/react-dom.js"></script>      <script src="https://cdnjs.cloudflare.com/ajax/libs/babel-core/5.8.23/browser.min.js"></script>    </head>    <body>      <div id="example"></div>      <script type="text/babel">        ReactDOM.render(          <h1>Hello, world!</h1>,          document.getElementById('example')        );      </script>    </body>  </html>

    A sintaxe XML dentro do JavaScript é chamada JSX ; vá para a JSX syntax para aprender mais sobre ele. A fim de traduzi-lo para baunilha JavaScript usamos <script type = "text / babel "> e incluem Babel para realmente executar a transformação no navegador.

    Arquivo separado

    Seu React código JSX pode viver em um arquivo separado . Crie o seguinte

    src/helloworld.js.

    ReactDOM.render(    <h1>Hello, world!</h1>,    document.getElementById('example')  );

    Em seguida, referenciá-lo a partir de helloworld.html:

    <script type="text/babel" src="src/helloworld.js"></script>

    Observe que alguns navegadores ( Chrome , por exemplo ) não poderá carregar o arquivo a menos que seja servido através de HTTP.

    Offline Transform

    Primeiro instale as ferramentas de linha de comando Babel ( requer npm ) :

    npm install --global babel-cli  npm install babel-preset-react

    Em seguida, traduzir o seu src/helloworld.js arquivo de JavaScript simples :

    babel --presets react src --watch --out-dir build

    Nota:

    Se você estiver usando ES2015 , você vai querer usar também o babel-preset-es2015 package.

    O arquivo build/helloworld.js é gerado automaticamente sempre que você fizer uma mudança. Leia o Babel CLI documentation para um uso mais avançado.

    ReactDOM.render(    React.createElement('h1', null, 'Hello, world!'),    document.getElementById('example')  );

    Atualize seu arquivo HTML como abaixo:

    <!DOCTYPE html>  <html>    <head>      <meta charset="UTF-8" />      <title>Hello React!</title>      <script src="build/react.js"></script>      <script src="build/react-dom.js"></script>      <!-- No need for Babel! -->    </head>    <body>      <div id="example"></div>      <script src="build/helloworld.js"></script>    </body>  </html>
